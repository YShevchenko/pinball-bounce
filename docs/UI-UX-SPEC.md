# Pinball Bounce UI/UX Specification

## Design Philosophy
- **Voodoo-Style Minimalism:** Absolutely zero UI clutter. The game starts immediately on launch.
- **Visual Hierarchy:** Bright, high-contrast primary objects against clean, muted backgrounds.

## Screens
1. **Title/Menu:** Massive "PLAY" button. Settings icon.
2. **Core Game Loop:** The main active gameplay area.
3. **Game Over / Victory Modal:** Shows score/level, a "Next/Restart" button, and triggers the interstitial ad logic.
